# AI Waterborne Disease Risk Prediction

## Overview

AI Waterborne Disease Risk Prediction is a machine learning-based web application developed to assess the risk of waterborne diseases using water quality and environmental parameters.

The system predicts whether a water source falls under LOW, MEDIUM, or HIGH risk categories and provides possible disease indications and health recommendations.

## Features

* Waterborne disease risk prediction using Machine Learning
* Risk levels: LOW, MEDIUM, HIGH
* Water quality assessment
* Health recommendations based on risk level
* Interactive Flask web interface
* State-wise and water-body-wise analysis

## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Scikit-Learn
* HTML
* CSS

## Machine Learning Model

* Fine-Tuned Decision Tree Classifier
* Multiclass Classification
* Feature Encoding using LabelEncoder
* Train-Test Split Evaluation

## Input Parameters

* Water Body Type
* State Name
* Temperature
* Dissolved Oxygen
* pH
* BOD
* Rainfall

## Output

* Risk Level Prediction
* Possible Diseases
* Water Quality Status
* Health Recommendations
